Atom Docset Generator
=====================

Generate [Atom][1] docset for [Dash][2]


Requirement
-----------

* httrack
* Python3


Usage
-----

```
$ brew install httrack  # If it is not installed
$ pip install -r requirements.txt
$ make
```
